# Customer Segmentation and Data Analysis

## Overview
This project analyzes mall customer data to understand customer demographics and spending behavior. The analysis helps identify patterns between income, age, gender, and spending score.

## Dataset
Mall Customer Segmentation Dataset (CSV)

## Tools and Technologies
Python  
Pandas  
Matplotlib  
Seaborn  
Google Colab

## Analysis Performed
Data loading and exploration  
Customer age distribution visualization  
Income vs spending score analysis  
Gender distribution analysis  

## Key Insights
Most customers fall between the age group of 25–40.  
Higher annual income often correlates with higher spending scores.  
Gender distribution shows variation in spending behavior.
